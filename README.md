## Quick start

Open `dist/index.html` in your browser. No install needed.

## Customizing the template

All customization lives at the top of `dist/assets/css/style.css`.
Edit only the values inside the `:root` block — everything else adjusts automatically.

### 🎨 Brand colors

```css
--primary: #18181b; /* buttons, links, accents */
--primary-foreground: #fafafa; /* text ON primary — keep white or black */
```

### 🌑 Page background & text

```css
--background: #ffffff; /* page background */
--foreground: #09090b; /* main body text */
--muted-foreground: #71717a; /* subtle text, labels, captions */
```

### 📦 Surfaces

```css
--secondary: #f4f4f5; /* cards, secondary buttons */
--accent: #f4f4f5; /* hover surfaces */
--border: #e4e4e7; /* borders and dividers */
--input: #e4e4e7; /* form input borders */
```

### 🔤 Font

```css
--font-sans: "Inter", sans-serif;
/* Any Google Font name works — update the <link> in index.html to match */
```

### 🔲 Border radius

```css
--radius: 0.5rem;
/* 0 = sharp edges · 1rem = very round */
```
