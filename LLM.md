## About the template

- Built with plain HTML + Tailwind CSS
- No framework, no build step required
- Open `dist/index.html` directly in the browser to preview
- All customization lives in `dist/assets/css/style.css` inside the `:root` block

---

## CSS variables (the only things users need to touch)

| Variable               | Purpose                                 |
| ---------------------- | --------------------------------------- |
| `--primary`            | Buttons, links, accents                 |
| `--primary-foreground` | Text ON primary (keep light/dark)       |
| `--background`         | Page background                         |
| `--foreground`         | Main body text                          |
| `--muted-foreground`   | Subtle text, labels, captions           |
| `--secondary`          | Cards, secondary buttons                |
| `--accent`             | Hover surfaces                          |
| `--border`             | Borders and dividers                    |
| `--input`              | Form input borders                      |
| `--font-sans`          | Font family (any Google Font)           |
| `--radius`             | Border radius (0 = sharp, 1rem = round) |

---

## How to help the user

- If they want to **change colors** → guide them to edit `--primary` and `--background` in `:root`
- If they want to **change the font** → update `--font-sans` and the Google Fonts `<link>` in `index.html`
- If they want to **change border radius** → edit `--radius`
- If they want to **edit content/layout** → guide them to `dist/index.html`
- If they paste their code → review it and suggest clean, minimal edits
- If they're stuck → ask what they're trying to achieve visually, then guide step by step
